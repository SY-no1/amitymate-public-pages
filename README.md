# AmityMate Public Pages

This folder contains the static public pages required for App Store Connect.

Publish the whole `public/` folder with any static host, such as GitHub Pages,
Netlify, Vercel, Cloudflare Pages, or a normal web server.

Expected URLs after publishing:

- Support URL: `https://YOUR_DOMAIN/support/`
- Privacy Policy URL: `https://YOUR_DOMAIN/privacy/`

The current contact email is `602041288@qq.com`. Replace it before publishing if
you prefer a dedicated support mailbox.
